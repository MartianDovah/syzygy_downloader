## 📖 プロジェクト概要

現代のWebページは動的レイアウトが複雑化しており、従来の「名前を付けて保存」や「コピー＆ペースト」でリソースを抽出することは非常に面倒になっています。**Syzygy** は、この問題を解決するために誕生しました。

その核心的な位置づけは、非常にスムーズな**「ドラッグ＆ドロップ式ウェブ素材収集ツール」**です。実行時の適応保護ロジックと安定性維持ロジックが組み込まれており、様々な複雑なフロントエンド環境でも安定して動作します。簡単な「ドラッグ」操作で、見たままのリソースキャプチャを実現します。

## ✨ 主な機能

### 🪐 1. 革新的な力学ベース重力エンジン (Dropzone フロート)
Syzygyは従来のリスト型の配置を完全に捨て、ウェブページに惑星のパートナーを追加します。自由にドラッグしたり拡大したりでき、キャプチャしたリソース（衛星ノード）と衝突し、絡み合います。
* **数学的重力場レイアウト**: フェルマーの螺旋と黄金角に基づいて衛星ノードの初期分布を計算し、視覚的な密度を均一にします。
* **スムーズな重力遷移**: ドラッグやフリック時には、EMA（指数移動平均）速度判定と一次ローパスフィルタを使用し、物理的な吸着や遠心力のエフェクトをスムーズにシミュレートします。
* **衝突と弾性境界**: 適応型の衝突回避とビューポート端での弾性ソフトバッファにより、複数のタスクが重なったり画面外に出たりするのを防ぎます。

![](assets/操作演示.gif)

### 🎬 2. Web動画のディープキャプチャ
通常の動画でも、特殊な処理で分割された複雑な動画でも、`Shift`キーを押しながらドラッグするだけで、バックグラウンドで自動的にトラフィック解析を行い、実際のダウンロードURLを正確に復元して抽出します。

![](assets/视频下载.gif)

### 🖼️ 3. 多形式・高解像度画像と背景画像の抽出
高解像度の元画像抽出をサポートし、複雑なページ階層を透過して、CSSの疑似要素や背景属性に隠された画像リソースを抽出します。

![](assets/图片下载.gif)
![](assets/GIF捕获.gif)

### 📊 4. 構造化テーブルとテキストの抽出
ウェブページ上の表やテキストをドラッグするだけで、構造化データや価値の高いテキスト情報を自動的に抽出します。面倒なコピー操作は不要です。

![](assets/表格捕获.gif)

### 📚 5. Webコミックと縦長画像の抽出 (ジェスチャー切り替え)
Canvasや複数ネストされた特殊な読書サイトでは、空中で**マウスを素早く左右に振る（フリックする）**だけでジェスチャーアルゴリズムがトリガーされ、候補となるリソースオブジェクトを順番に切り替えて正確にロックします。

![](assets/网页漫画捕获.gif)

### 💻 6. ピュアなソースコードキャプチャ
現在のページや特定のiframeのクリーンなソースコードをワンクリックでキャプチャしてフォーマットします。面倒な開発者ツールを開くことなく、構造を直接分析できます。

![](assets/网页源码.gif)

### 🛡️ 7. 広告とトラッキングスクリプトのブロック
Webリクエストに対して多重ルールフィルタリングを自動的に適用し、広告やトラッキングスクリプトをブロックして、キャプチャしたメディアソースのクリーンさを保ちます。

![](assets/广告屏蔽.gif)

### 💾 8. ワンクリック高速ローカルダウンロード
ブラウザのネイティブダウンロードインターフェースを直接呼び出します。キャプチャが完了したら、衛星ノードをクリックするだけで、リソースがローカルに安全に保存されます。

![](assets/下载演示.gif)

## 🔧 インストールと構成

1. **リリースパッケージのダウンロード**:
   プロジェクトのリリースには以下の2つのファイルが含まれています。両方をダウンロードしてください：
   - **`Syzygy_Extension.zip`**（ブラウザ拡張機能）
   - **`Syzygy_Server.exe`**（サーバー側プログラム）

2. **拡張機能のインストール**:
   - ダウンロードした `Syzygy_Extension.zip` を解凍し、任意のフォルダに配置します（誤って削除されない場所を推奨します）。
   - ブラウザを開き、拡張機能管理ページ（Chrome の場合は `chrome://extensions/`）に移動します。
   - 画面右上の **「デベロッパー モード」**（Developer mode）を有効にします。
   - **「パッケージ化されていない拡張機能を読み込む」**（Load unpacked）をクリックし、解凍した拡張機能フォルダを選択してインストールします。

3. **サーバーの構成と接続**:
   - **`Syzygy_Server.exe`** をダブルクリックして実行します。利用規約を確認し、同意すると、デフォルトで **「試用モード」** に入ります。
   - サーバー側ターミナルのメインメニューに表示される **`AccessToken`** をコピーします。
   - ブラウザに戻り、Syzygy 拡張機能アイコンを右クリックして **「オプション」**（Options）を開きます。
   - コピーした `AccessToken` を **「内网通行证 (Access Token)」** フィールドに貼り付けます。
   - オプション画面の **「后端基站通信地址 (Server URL)」** の IP とポートが、サーバー側ターミナルに表示されているものと一致していることを確認します。
   - 確認後、**「保存」** をクリックします。
   - サーバー側ターミナルに戻り、数字の `1` を入力してサービスを起動します。
   - ブラウザに戻り、拡張機能インターフェースの **「サーバー接続インジケーター」** が **緑色** に点灯すれば、接続完了です。リソースのキャプチャとダウンロードを開始できます。

## 🎮 操作ガイド

- **要素のキャプチャ**: `Shift` キーを押しながら、マウスでウェブページ上の要素をクリックしてドラッグします。
- **リソースのダウンロード**: キャプチャした要素を Dropzone フロート窓にドラッグ＆ドロップすると、ダウンロードが開始されます。
- **候補オブジェクトの切り替え**: 同一箇所で複数の要素をキャプチャした場合、マウスを空中（左右）で**素早く振る（フリックする）**ことで、候補となるリソースを切り替えられます。
- **キャッシュのクリア**: ショートカットキー `Shift + Alt + X` で、現在のウェブサイトのキャッシュと履歴を消去します。
- **デバッグキャプチャモード**: ショートカットキー `Ctrl + Shift + D` を押すとデバッグパネルが開き、キャプチャされた要素の数が正しいかどうかを確認できます。

## 🔒 声明と技術的保護

**1. クラッキングや権限昇格は含みません**
Syzygyには、デジタル著作権管理（DRM：Widevine、FairPlayなど）を解読するためのコードや、アカウントの権限昇格、VIPペイウォールの回避機能は一切含まれていません。本拡張機能は、ブラウザの準拠APIフレームワーク内で、ウェブページ上で暗号化されずに送信されているデータストリームを統合・抽出するのみであり、面倒な手動パケットキャプチャの代わりとなるものです。

**2. 現在の「クローズドソース」および技術的保護に関する説明**
オープンソースでの共有を好む開発者として、このプロジェクトを一時的にクローズドソースにする決断は非常に難しいものでした。これは主に以下の考慮事項に基づいています。
* **悪用とAPIスクレイピングの防止**: プラグインには、高度なリソース取得技術とバイパス技術が組み込まれています。完全にオープンソース化すると、悪意のある業者によって大規模な商用スクレイピングツールとして再パッケージ化されやすくなり、ターゲットサイトの防御強化や、バックエンドサービスへの悪意のある大量アクセスを引き起こす可能性があります。
* **コア技術の保護とリバースエンジニアリング防止**: 高度な防御が施されたWebページに対応するため、プラグイン内には一連の実行時保護メカニズム（ホストからの探知を防ぐサンドボックス強化、汚染防止メカニズム、DOMの動的自己修復ガードなど）を実装しています。完全にオープンソース化すると、これらの防御が意図的にクラッキングされたり、転売されたりして、最終的に悪用される危険性があります。

著作権を効果的に保護できる方法が見つかり次第、オープンソース化を検討する予定です。皆様のご理解をお願い申し上げます。
